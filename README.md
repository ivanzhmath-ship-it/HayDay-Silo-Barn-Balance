# HayDay-Silo-Barn-Balance
Calculates the ideal amount of nails, desks etc. depending on current amount.

Firstly give it number of your farms (1 for one farm, n for n farms), then enter how much of each material you own (your first farm will be the most prioritized, your last farm will be less prioritized). 

As result, you'll recieve the ideal amount of each material in same order as you entered farms and materials.
